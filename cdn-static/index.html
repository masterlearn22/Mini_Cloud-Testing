Prinsip IDEAL Cloud Computing

  • Isolated — Setiap container berjalan dalam namespace terisolasi
  • Democratic — Semua user mengakses melalui satu entry point (LB port 80)
  • Elastic — Jumlah worker dapat ditambah/dikurangi sesuai beban
  • Adaptive — Sistem merespons kegagalan node secara otomatis (fault tolerance)
  • Less-coupled — Setiap layanan independen, dapat di-update tanpa mematikan keseluruhan sistem
